# Differentiable Support Functions

## Reference

> S. Park, J. Lee, and D. J. Lee, "Shape Abstraction via Marching Differentiable Support Functions," CVPR 2025

## DSF Fitting
Implementation for DSF Fitting for a mesh.
1. Input obj file
```
obj/fitting/*.obj
```
2. Run 'main_fitting.m'
3. Output obj file
```
res/*.obj
```
